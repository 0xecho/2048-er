About page

A remake of the 2048-game but with a competitive programming twist.

How to play

  • Your code accepts the current board as input
  • Your code must return a possible move to be made
  • When two tiles with the same number touch, they merge into one!
  • The game ends when there are no more possible moves.

Rules

  • The game is played on a 4x4 grid
  • The game ends when the total time elapsed is more than 5 minutes
  • The game ends when there are no more possible moves
  • The game ends when you have made 20,000 moves

Scoring

  • The score is the sum of the tiles multiplied by log 2

How to submit

  • The code must be submitted as a single python file on the submission page
