Welcome to 2048'er.

A competitive programming website where you can battle on the game 2048 using python.

Download this boilerplate.py code to get started quickly.

Problem Description

Write a code that accepts a 4x4 grid as an input and plays the 2048 game.

The game is played by moving the tiles around the grid.

Input

The new state of the board after each move. one round is four lines of input each line has four numbers separated by a space
example
0 0 4 0
2 0 0 4
2 2 2 2
1024 0 0 4

Output

A number from 0 - 3 represent the direction to move the grid. 0 - left 1 - up 2 - right 3 - down

The game will accept your input until EOF is reached or 1000 states have passed, which ever one comes first. The score is calculated by using the following snippet:
score = 0
for row in BOARD:
     for col in row:
         score += col * log2(col)
